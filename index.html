signingsavvy.api.search(sign: str) → dict¶

Function to pull one or more signs through a search result.

Note

@app.route(“/search/<sign>”)

If only one sign is found, the information is returned directly. If more signs are found, a list of results are returned.

Parameters

sign (str) – The word/term to search for.

Returns

Response containing a response of matching signs or a response containing information from the sign in the event that only a single sign is found.

Return type

dict
